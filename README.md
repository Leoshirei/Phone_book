# Phone Book

Phone Book is a simple CLI application written in Python for managing contacts.
Data is stored in a JSON file.

## Features

- Add new contact (name, phone number, email)
- Delete existing contact
- Update contact information
- Save and load data from JSON file

## Technologies

- Python
- JSON
- OOP

## Purpose

This project was created for learning backend fundamentals such as:
- object-oriented programming
- data persistence
- project structure
